## Lab 2: Cleaning & Normalization

**Мета:** Побудова детермінованого пайплайну для очищення та нормалізації перекладеного українського NLI-датасету.

**Як запустити:**
1. Встановіть залежності.
2. Запустіть пайплайн: `python src/lab2.py`
3. Результати будуть збережені у `data/processed_v2.csv`, а крайові випадки згенеровані у `tests/edge_cases.jsonl`.
